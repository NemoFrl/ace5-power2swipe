# FlashNote（一键闪记）

长按电源键即可触发 ColorOS 一键闪记功能，适用于无AI键的机型。

## 功能

- 长按电源键约 1 秒，自动触发 ColorOS 一键闪记
- 基于 KernelSU 模块，开机自启，无需额外配置
- 通过直接调用系统 `ColorDirectService` 服务触发，响应迅速

## 安装

1. 下载 [最新版本](https://github.com/NemoFrl/coloros-flashnote/releases/latest) 的 zip
2. 在 KernelSU 管理器中点击「安装模块」，选择下载的 zip 文件
3. 重启设备
